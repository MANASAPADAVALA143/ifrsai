## 🚀 Deploy from GitHub

### Frontend (Vercel)

1. Go to [https://vercel.com](https://vercel.com)
2. Click "Import Project"
3. Connect GitHub account
4. Select `MANASAPADAVALA143/ifrsai`
5. Root Directory: `frontend`
6. Add environment variables:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `NEXT_PUBLIC_API_URL`
7. Deploy!

### Backend (Render/Railway)

1. Create account on Render.com or Railway.app
2. Connect GitHub
3. Select repository
4. Add `ANTHROPIC_API_KEY` as environment variable
5. Deploy command: `uvicorn app:app --host 0.0.0.0 --port $PORT`

## 🔧 Troubleshooting

### "Permission denied" when pushing

```bash
# Set up SSH or use personal access token
# Go to: https://github.com/settings/tokens
# Create token, use it as password
```

### "Merge conflict"

```bash
# Pull latest changes first
git pull origin main

# Resolve conflicts manually
# Then commit and push
```

### "Git not recognizing changes"

```bash
# Check .gitignore isn't excluding files
# Check file was actually modified
git status -u
```
